Accessibility Guide

Font Size Adjustment

Use the slider in the Accessibility Features dropdown to increase or decrease the font size. This affects all text on the website for easier reading.

Dyslexic Font

Toggle the Dyslexic Font switch to enable a font that may be easier to read for users with dyslexia.

High Contrast Mode

Toggle the High Contrast switch to enable a color scheme with stronger contrast. This can help users with visual impairments to better distinguish content.

Simplified Layout

Enable the Simplified Layout option to reduce visual clutter and present content in a more straightforward manner. This can be helpful for users with cognitive disabilities or those who prefer a less complex interface.

Colorblind Mode

Choose a colorblind mode from the dropdown menu to adjust the website's colors for different types of color vision deficiencies.

Voice Control

Enable Voice Control to navigate the website using voice commands. Say "Read [section name]" to have the content read aloud.

Keyboard Navigation

Our website is fully navigable using a keyboard. Use the Tab key to move between interactive elements, and Enter or Space to activate them.

Saving Preferences

Your accessibility settings are automatically saved and will be applied on your future visits to our website.
